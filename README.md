# simpleGoDatabase
is a commandline based for CRUD mysql database.

written in go

## How To Use
```bash
Usage: crud [global options]

Global options:
        -h, --help   show this menu
            --test   run action
        -r, --read   Read data from table (default: true)
        -u, --update Update data from table
        -d, --delete Delete data from table
```
